User Management System

A simple CRUD (Create, Read, Update, Delete) application built using Next.js and React.js.
This project allows users to add, view, update, and delete user records dynamically using API routes.
The application uses Next.js API Routes, React Hooks, React States, components, NextRespose, async, path, fs, JSON Methods, JSON-based storage.

Features

•	Add New Users

•	View All Users

•	Update Existing Users

•	Delete Users

•	Confirmation Popup before deletion

•	Responsive UI

This project helps in understanding:

•	Frontend and Backend integration

•	API handling

•	State management

•	CRUD workflows

•	Dynamic rendering in React

Technologies Used

Frontend

•	Next.js

•	React.js

•	CSS

Backend

•	Next.js API Routes

•	Node.js File System (fs)

Data Storage

•	JSON File (users.json)
